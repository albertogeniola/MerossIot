Ms100Sensor¶

class meross_iot.controller.subdevice.Ms100Sensor(hubdevice_uuid: str, subdevice_id: str, manager, **kwargs)¶

This class maps the functionality offered by the MS100 sensor device. The MS100 offers temperature and humidity sensing. Moreover, this device is capable of triggering settable alerts.

property last_sampled_humidity¶

Exposes the latest sampled humidity, in %. If you want to refresh this data, call async_update to force a full data refresh.

Returns

The latest sampled humidity grade in %, if available

property last_sampled_temperature¶

Returns the latest sampled temperature in Celsius degrees. If you want to refresh this data, call async_update to force a full data refresh.

Returns

The latest sampled temperature, if available, in Celsius degree

property last_sampled_time¶

UTC datetime when the latest update has been sampled by the sensor

Returns

latest sampling time in UTC, if available

property max_supported_temperature¶

Minimum supported temperature that this device can report

property min_supported_temperature¶

Maximum supported temperature that this device can report

Returns

float value, maximum supported temperature, if available
